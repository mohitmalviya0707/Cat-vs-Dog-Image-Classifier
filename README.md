# 🐶🐱 Cat vs Dog Image Classifier

A Deep Learning project that classifies images as **Cat 🐱 or Dog 🐶** using a Convolutional Neural Network (CNN) with a simple and interactive Streamlit UI.

---

## 🚀 Features

* 📤 Upload any image
* 🤖 AI-based prediction (Cat or Dog)
* ⚡ Fast and lightweight Streamlit app
* 📊 Model trained using CNN

---

## 🛠 Tech Stack

* Python
* TensorFlow / Keras
* Streamlit
* NumPy, Pandas, Matplotlib, Seaborn

---

## 📂 Dataset

* Kaggle Cats vs Dogs Dataset


## 📈 Model Performance

* Accuracy: ~98% (can be improved with tuning)
